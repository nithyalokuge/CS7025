Audio Editing

Project Overview

Record, edit and apply audio effects to personal voice.

Objective

The goal is to develop practical skills in recording, editing, and processing voice audio while also exploring the use of different audio effects.

Key Requirements

  • Record personal voice and save it in an uncompressed format (WAV or AIFF).
  • Clean and edit the recording using Audacity and save it as a MONO UNCOMPRESSED PCM audio file (48kHz, 24-bit).
  • Apply TWO realistic and TWO abstract/special effects treatments to the recording using audio effects such as Reverb, Delay, Modulation, etc.
  • Save each modified file as a STEREO UNCOMPRESSED PCM audio file (48kHz, 24-bit).

Outcome

Available by January 25, 2025
